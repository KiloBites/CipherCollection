Keep Talking and Nobody Explodes Mod Collon CM

On the Subject of the Collon CM

A machine that knows no bounds when it comes to ciphers.

Create a key using the keyword on screen 1 and the Boolean Expression on screen A. Make sure to replace any Js with Is. Rearrange the key into a 5x5 matrix.

Concatenate the encrypted word and screen 2 in that order. Split the newly created string into letter pairs.

Do the following for each letter pair:

  • Find the 1st letter in the pair in the matrix and use that as the row.
  • Find the 2nd letter in the pair in the matrix and use that as the column.
  • The intersection of the 2 letters will give you a decrypted letter.

Finally, if any of the letters on screen 3 match at the same position as any of the letters of your encrypted word, replace that letter with a J. You should now have an decrypted word.

Example

Encrypted Word: YGQZRGS
Screen 2: AOBDTZN
Key: ABCFKMNOPQRSUVWXYZDELIGHT
Screen 3: ZPLEEAE

YG -> Z
QZ -> O
RG -> U
SA -> R
OB -> N
DT -> E
ZN -> Y
ZOURNEY + ZPLEEAE -> JOURNEY
Decrypted Word: JOURNEY
