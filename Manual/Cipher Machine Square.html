Keep Talking and Nobody Explodes Mod Square CM

On the Subject of the Square CM

A machine that knows no bounds when it comes to ciphers.

From the Shape Cipher module by TracksJosh.

Create a key using the keyword on screen 1 and the Boolean Expression on screen A. Make sure to replace any Js with Is. Rearrange the key into a 5x5 matrix.

Split the encrypted word into pairs of letters. If there is an odd amount of letters, leave the last letter as the odd letter.

For each pair of letters, do the following:

  • Find the pair of letters in the matrix.
  • If the pair of letters share the same row or column, rotate the positions of the 2 letters in the matrix by 180°, replacing each pair of letter with the letters that you ended up on.
  • Otherwise, replace each letter with the letter on the same column but in the row of the other letter in the pair.

Finally, if any of the letters on screen 2 match at the same position as any of the letters of your encrypted word, replace that letter with a J. You should now have an decrypted word.
